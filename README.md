## Installation

```sh
# Using npm
npm install stl-kit
# Using yarn
yarn add stl-kit
# Using pnpm
pnpm add stl-kit
```

---

## Usage

Import the desired data structure:

```ts
// Using ES6 import
import { LinkedList } from 'stl-kit'
// Using CommonJS require
const { LinkedList } = require('stl-kit')
```
